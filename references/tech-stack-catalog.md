# 技术栈分类索引

为 AskUserQuestion 提供选项来源。所有选项平等展示，不标"推荐"。

## 编程语言

| 语言 | 适用场景 |
|------|---------|
| TypeScript/JavaScript | Web 全栈、前端、Node.js 后端 |
| Python | 后端 API、数据处理、AI/ML |
| Go | 高性能后端、CLI 工具、微服务 |
| Rust | 系统级工具、高性能服务、CLI |
| Java/Kotlin | 企业级后端、Android |
| Swift | iOS/macOS 原生 |
| Dart | Flutter 跨平台移动端 |

## 前端框架

| 框架 | 描述 |
|------|------|
| React | 组件化 UI 库，生态最大 |
| Vue 3 | 渐进式框架，上手快 |
| Next.js | React 全栈框架，SSR/SSG/API Routes |
